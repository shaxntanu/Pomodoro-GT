# Pomodoro Focus

A sleek, minimalist Pomodoro timer built with Next.js and TypeScript. Designed to help you stay focused and productive with a beautiful starry night interface.

## Features

- **Customizable Timer** - Adjust focus time, short breaks, long breaks, and session counts to match your workflow
- **Task Management** - Create, track, and complete tasks alongside your Pomodoro sessions
- **Statistics Dashboard** - Track your focus time, completed sessions, streaks, and daily progress
- **Audio Notifications** - Configurable sounds with volume control for session transitions
- **Keyboard Shortcuts** - Quick controls with Space, R, S, T, and Escape keys
- **Auto-Start Options** - Automatically begin breaks or focus sessions when the previous one ends
- **Responsive Design** - Optimized for desktop, tablet, and mobile devices
- **Dark Theme** - Easy on the eyes with an animated starfield background
- **PWA Support** - Install as a standalone app on your device
- **Local Storage** - Your settings, tasks, and stats persist across sessions

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion
- Zustand (state management)
